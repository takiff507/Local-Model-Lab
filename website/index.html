Questions

Before you download

Does Local Model Lab work offline?

Yes. After a model and any required image backend are downloaded, inference runs locally. Internet access is still needed to download new model files.

Do I need a powerful GPU?

No for lightweight text models. A compatible GPU improves speed and is strongly recommended for image generation and larger models.

Where are my models and images stored?

They are stored in Local Model Lab folders under your Windows user-data directory. The app provides buttons to open both locations.

Why might Windows show a security warning?

Early unsigned releases may trigger Microsoft SmartScreen because the publisher has not yet established signing reputation. Verify the release checksum and download only from the official repository.

Can I import a model I already have?

Yes. Supported local GGUF, SafeTensors, CKPT, and BIN files can be imported from Model Library.
