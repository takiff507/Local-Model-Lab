# Local Model Lab branding

Place the final master logo here as:

`branding/logo-master.png`

Requirements:
- 1024 x 1024 pixels
- transparent PNG
- square canvas
- keep important artwork inside the central 80%
- no background color baked into the image

This one master will be exported to:
- `website/assets/logo.png` for the website header
- `website/assets/favicon.ico` and PNG favicon sizes
- `public/logo.png` for the desktop UI
- `build/icon.ico` for Electron, Windows taskbar, executable, and installer

Do not manually resize the master. Add the original file and regenerate all outputs together.
